# Solutions' Lists
Aplikacja do wspólnego rozwiązywania list zadań z matematyki i fizyki, inspirowana Dyskiem Google.
Rozwiązania można zapisywać w TeXu, rysować na płótnie lub wysyłać w formie plików graficznych.
